## ðŸ›  Features

- Download historical stock data from Yahoo Finance.
- Display the dataset preview.
- Visualize closing price charts using Matplotlib.
- Display candlestick charts with volume using `mplfinance`.
- Technical indicators:

  - Bollinger Bands
  - RSI (Relative Strength Index)

- Compare actual vs predicted stock prices in interactive charts.
