This project is a **Stock Price Prediction application** built using **Python**, **Streamlit**, and **LSTM (Deep Learning)**.
The app allows users to analyze historical stock data, visualize trends, and predict future stock prices using machine learning.

---

## Features

- Fetch real-time stock data using **Yahoo Finance API**.
- Visualize stock data with **line charts** and **candlestick charts**.
- Predict future stock prices using **LSTM neural networks**.
- Interactive and user-friendly interface built with **Streamlit**.
- Provides insights into stock trends and performance.

---

## Technologies Used

- **Python 3.10+**
- **Streamlit** for web app interface
- **yfinance** for stock data
- **pandas** for data manipulation
- **numpy** for numerical computations
- **matplotlib & mplfinance** for visualizations
- **TensorFlow & Keras** for LSTM model
