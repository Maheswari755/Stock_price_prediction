Overview

This is an interactive Stock Price Prediction Web Application built using Python, Streamlit, and LSTM deep learning models. The app allows users to select a stock from a predefined list, visualize historical stock data, view technical indicators like moving averages and RSI, and predict future stock prices using an LSTM model.

The project combines data visualization, machine learning, and web deployment to provide an end-to-end solution for stock market analysis.

Features

Stock Selection: Choose from popular stock symbols such as GOOG, AAPL, MSFT, TSLA, AMZN, META, and more.

Historical Data: Fetch historical stock prices using Yahoo Finance (yFinance).

Visualizations:

Closing Price Chart

Candlestick Chart

Moving Averages & Bollinger Bands

Relative Strength Index (RSI)

LSTM Stock Prediction: Predict future closing prices using a trained LSTM model.

Interactive Web App: Built with Streamlit for a smooth, real-time experience.

Custom Date Range: Select any start and end date to analyze stock data.
